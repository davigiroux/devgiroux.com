## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Content:** MDX with gray-matter
- **Search:** Fuse.js
- **Analytics:** Vercel Analytics
- **Comments:** Giscus
- **Deployment:** Vercel

## Design

- **Theme:** Electric Violet (#8b5cf6)
- **Font:** Lexend Deca
- **Style:** Dark mode default, minimal, clean
